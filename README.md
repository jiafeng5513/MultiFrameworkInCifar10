Multi framework test in Cifar-10
=====

1. 使用Cifar-10数据测试多种深度学习框架<br>
2. 额外包GraphViz pydot

#### 环境说明
1. Windows 10,Anaconda3,python3.6.1,Keras,PyTorch,Tensorflow,CNTK
2. 使用CNTK时如果提示找不到DLL,请尝试删除java环境变量.
